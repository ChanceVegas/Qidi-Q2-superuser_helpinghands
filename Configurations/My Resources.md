## How to use these links
1. Click the repo link and check the README for install or usage instructions.  
2. Look for branches or forks if you need alternate versions.  
3. Use issues or discussions in each repo to ask questions or report problems.

---

## Quick tips
- **Bookmark** the KAMP integration repo if you plan to modify macros.  
- **Search** each repo for `README`, `INSTALL`, or `CHANGELOG` before copying configs.  
- **Backup** your current printer config before applying changes.
