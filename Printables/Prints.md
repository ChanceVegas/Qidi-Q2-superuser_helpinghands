# Recommended Prints for the Qidi Q2

These are the prints I use on my own Q2. The **only model with different slicer settings is the riser** (noted below).

---

## Purge Bin

I don’t use multicolor often, so I went with a small, simple purge bin that fits cleanly in the chamber.

- [Purge Bin](https://www.printables.com/model/1390449-qidi-q2-poop-chutebucket/files)

---

## Riser

This riser works great — I was able to fit three sides on a single plate and run it overnight. I also printed the tabs in a different color.  
**This is the only print that uses different slicer settings.**

- [Riser](https://www.printables.com/model/1443536-qidi-q2-riser)  
- [Riser Settings](https://github.com/Camden-Winder/Qidi-Q2-superuser/blob/main/Printables/Riser%20Settings.txt)

---

## Better Handle

The stock Qidi handle isn’t great. This replacement uses the factory screws, which is a huge bonus.  
**Note:** Enable adaptive layer height for this print.

- [Door Handle](https://www.printables.com/model/1647312-yet-another-qidi-q2-door-handle)

---

## Screen Cover

Does it do anything functional? Not really. Does it match the black‑and‑white theme I have going on? Absolutely.

- [Screen Cover](https://www.thingiverse.com/thing:7329016)

---

## Cardboard Spool Adapters

These help cardboard spools run smoother inside the enclosure. In my experience, they work well.

- [Spool Adapters](https://makerworld.com/en/models/988003-universal-cardboard-spool-adapter-ring?from=search#profileId-963058)

---

## Filament Clip

Great for holding filament rolls closed during storage or when drying in the box.  
I printed mine in PETG and scaled the model up by **5%**.  
Use **0.16 mm layer height** as the creator recommends.

- [Filament Clip](https://makerworld.com/en/models/1303141-clip-mate-filament-clip?from=search#profileId-1335869)
